# 📝 BasicTools –
1-Python Not Defteri & Otomatik Tweet Atıcı

Bu proje, komut satırı üzerinden çalışan basit bir not defteri uygulamasıdır.  
Ayrıca, notların ilk satırını **belirli bir saatte otomatik olarak Twitter'da paylaşma** özelliği sunar (Selenium ile).

---

## 🚀 Özellikler

- Yeni not ekleme
- Mevcut notları listeleme
- İlk notu belirlenen saatte otomatik tweet atma (eğer Twitter hesabın Google Chrome'da açıksa)

---

## 🖥 Gereksinimler

- Python 3.x
- Google Chrome (Twitter'da oturum açık olmalı)
- ChromeDriver (tarayıcı sürümüne uygun)
